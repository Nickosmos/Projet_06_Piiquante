## Installation
***
About the installation. 
```
$ git clone https://github.com/Nickosmos/Projet_06_Piiquante.git
$ cd ../yourfolder/backend
$ nodemon server

$ cd ../yourfolder/frontend
$ npm install
$ npm start
```
